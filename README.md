# User-car-risk-predictor
A web app that predicts buyer risk for used car listings



# Dataset Note:

The dataset used for this project is Craigslist Cars and Trucks dataset from Kaggle.

Because the dataset is too large to upload to GitHub, the full dataset is not included in this repository; instead, here is the link to it. 
Dataset link:
https://www.kaggle.com/datasets/austinreese/craigslist-carstrucks-data
