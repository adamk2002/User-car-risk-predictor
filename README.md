# Used Car Purchase Decision using Risk Score

The objective of the project is to assist young buyers, students, or first-time car buyers by developing a machine learning decision-support tool that evaluates the potential risk of purchasing a used vehicle based on real Craigslist cars and trucks data.

The final purpose of the project is to create a risk score that classifies vehicles into meaningful buyer categories such as smart buy, fair deal, overpriced, high-mileage risk, and avoid.




# Dataset Note:

The dataset used for this project is the Craigslist Cars and Trucks dataset from Kaggle.
